# API Documentation

This section provides detailed documentation for the Agent Task Management System API.

## Endpoints

- **GET /tasks**: Retrieve a list of all tasks.
- **GET /tasks/{task_id}**: Retrieve details of a specific task.
- **POST /tasks**: Create a new task.
- **PUT /tasks/{task_id}/status**: Update the status of a task.

## Authentication

API access requires authentication. Please refer to the security guidelines for details on obtaining and using API keys.
